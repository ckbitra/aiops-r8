# Tools for Creating Workflows with AWS Icons

Here are common tools for creating workflows with AWS icons:

## Free / Online


| Tool                       | URL                                                                                                               | Notes                                                                                                     |
| -------------------------- | ----------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------- |
| **draw.io (diagrams.net)** | [draw.io](https://app.diagrams.net/)                                                                              | Free, no signup. AWS shape libraries (2017–2019), 3D and isometric. Use "More Shapes" → search "AWS".     |
| **Visual Paradigm Online** | [online.visual-paradigm.com](https://online.visual-paradigm.com/diagrams/features/aws-architecture-diagram-tool/) | AWS architecture diagram tool with 2019 AWS icons.                                                        |
| **Sketchboard**            | [sketchboard.io](https://sketchboard.io/)                                                                         | Real-time collaboration, hundreds of AWS icons. Search "aws2" for current icons. Export to PNG, SVG, PDF. |
| **Diagram Generator**      | [diagram-generator.com/aws](https://diagram-generator.com/aws)                                                    | AI-powered: describe architecture in text, generates diagrams with AWS icons.                             |


## Automated from AWS


| Tool            | URL                                 | Notes                                                                                               |
| --------------- | ----------------------------------- | --------------------------------------------------------------------------------------------------- |
| **Cloudviz.io** | [cloudviz.io](https://cloudviz.io/) | Connects to your AWS account, auto-generates diagrams from live resources, uses official AWS icons. |


## Desktop / Other


| Tool                       | Notes                                                                                                                |
| -------------------------- | -------------------------------------------------------------------------------------------------------------------- |
| **Lucidchart**             | AWS icon library; paid plans.                                                                                        |
| **Microsoft Visio**        | AWS stencils available; paid.                                                                                        |
| **AWS Architecture Icons** | [aws.amazon.com/architecture/icons](https://aws.amazon.com/architecture/icons/) – Official icons to use in any tool. |


## Recommended for This Project

- **draw.io** – Free, works in browser, supports AWS icons and workflows.
- **Diagram Generator** – Good if you want to generate diagrams from text descriptions.
